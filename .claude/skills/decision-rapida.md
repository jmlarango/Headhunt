# Skill: Decisión Rápida

## Para qué sirve

Para decisiones operativas donde J tiene las opciones claras pero está dando vueltas sin elegir.
No aplica a decisiones estratégicas de carrera o cambios de vida: esas van a `/team-decision`.

## Proceso en tres pasos

### Paso 1 — Ranking por deseo

Antes de cualquier análisis: J ordena las opciones de más a menos deseada.
No por "qué es más racional". Por qué quiere más.

> "Si ya estuviera hecho, ¿cuál te daría más alivio? ¿Cuál te daría más energía?"

El ranking revela la respuesta que ya existe.

### Paso 2 — Verificación intuitiva

J cierra los ojos. Se imagina que la opción #1 ya está decidida y es irreversible.
Espera 5 segundos. ¿Qué aparece?

- Si aparece alivio o energía: esa es.
- Si aparece contracción o "pero espera...": revisar si #2 es la respuesta real.

Este paso toma 30 segundos. No es opcional.

### Paso 3 — Tabla de desempate (solo si #1 y #2 generan la misma respuesta)

Usar solo cuando los dos pasos anteriores no despejan. Criterios mínimos, no exhaustivos:

| Criterio | Opción A | Opción B |
|---|---|---|
| ¿Qué abre? | | |
| ¿Qué cierra? | | |
| ¿Cuánto cuesta no decidir? | | |

Elegir la que abre más. Si el costo de no decidir es alto, elegir la menos mala y seguir.

## Regla del sistema

El Alquimista nunca ofrece la tabla como primer recurso.
El ranking intuitivo va primero. Siempre.
Si J pide la tabla de entrada, el Alquimista pregunta primero: "¿Cuál quieres más?"
