# Skill: Teoría de Juegos — Marco Analítico

## Cuándo aplicar

Cuando hay al menos dos actores con intereses propios, información parcial, y la acción de uno afecta el resultado del otro. Ejemplos: negociación salarial, aceptar o rechazar una oferta, posicionar una propuesta de coaching, decidir si esperar o actuar ante una vacante.

## Mapa de análisis

### 1. Identificar jugadores y payoffs

| Jugador | Objetivo real | Alternativa si no hay acuerdo (BATNA) |
|---|---|---|
| J | | |
| Contraparte | | |

El BATNA es el poder real. Quien tiene mejor alternativa afuera tiene más poder adentro.

### 2. Tipología del juego

**Suma cero**: lo que gana uno lo pierde el otro. Ejemplo: precio en una negociación salarial pura.
**Suma positiva**: existe un acuerdo donde ambos ganan más que sin acuerdo. Buscar si existe.
**Juego iterado**: habrá más rondas. La reputación importa tanto como el resultado inmediato.
**Juego de señalización**: una de las partes tiene información que la otra no tiene, y actúa para revelarla o esconderla.

### 3. Estrategias dominantes

Una estrategia es dominante si es la mejor respuesta independientemente de lo que haga el otro.
Si J tiene una estrategia dominante: ejecutarla sin vacilación.
Si no la tiene: buscar el equilibrio de Nash (dónde ninguno quiere desviarse unilateralmente).

### 4. Movimientos estratégicos

**Compromiso creíble**: hacer una promesa o amenaza que la contraparte crea porque es costoso incumplirla.
**First-mover advantage**: en algunos juegos conviene moverse primero para fijar el marco.
**Espera estratégica**: en otros, el que espera tiene más información.

### 5. Información asimétrica

¿Qué sabe J que la contraparte no sabe?
¿Qué sabe la contraparte que J no sabe?
¿Qué señales está leyendo la contraparte de las acciones de J?

## Salida del análisis

El sistema no produce una recomendación genérica. Produce:
1. La estrategia dominante de J, si existe.
2. El rango de acuerdo posible (zona ZOPA).
3. El umbral de reserva: el punto debajo del cual J no debe ceder.
4. La primera jugada recomendada con su justificación.
